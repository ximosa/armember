Installation & Activation

Steps For Installation And Activation

You can install plugin in following two ways. and ARMember - gm payment gateway plugin requires ARMember Plugin installed and active.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armembergroupmembership.
  2. In your WordPress installation directory, upload armembergroupmembership to wp-content/plugins/
  3. Activate the ARMember - Group Membership plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Group Membership Setup Wizard

Steps to Setup Group Membership For Your Website

Let's setup ARMember - Group Membership for your website :

  1. After activate addon, you can enable group membership for any plan. Group Membership plan enable/disable section will display in ARMember -> Manage Plans -> Add / Edit Plan -> Group Membership Settings.

    Please see below screen.

  2. Enter your maximum members limit in 'Maximum Members' option and enter value of your minimum members limit in 'Minimum Members' option. Select your purchase slot from 'Child user seat purchase slot'.

  3. After activating Group Membership for plan, configure your plan with 'Configure Plan + Signup Page'. After configure your setup, copy setup shortcode and add in your signup page. Now when you access your signup page it have option of how many users you want to purchase like below screen.

    In 'Sub User Selection' section it will display purchase seats according to your purchase slot selection for plan.

    You can change label of 'Sub User Selection' for your plan setup from 'Configure Plan + Signup Page'. Navigate 'Configure Plan + Signup Page' -> Other Options and there is option named 'Sub User Selection Label' from here you can change label value for your setup.

    For add summary text in your plan summary text, you can add it from your 'Configure Plan + Signup Page'. In 'Summary Text' option from your plan setup there is one option of '[SELECTED_CHILD]' which will replace with your selected child user.

  4. For child user signup, Enable 'Enable Child User Invitation?' option in your register page from 'Membership Shortcode' option. Please see below screen.

    Also you can change value of your child user invite code field label & child user invite code field placeholder from 'Membership Shortcode'. Please see below screen.

    Also admin user can view child users data with on click of child users counter in Group membership page.

  6. After signup with Group Membership, parent user can view their child user data with shortcode from 'Membership Shortcode' in their page. For add shortcode see below screen.

    This shortcode displays data of child user invitation code and added child users list like following screen.

Common Messages For Group Membership

Go to Common Messages tab at General Settings to configure Error messages and success messages.

To set default error message and success message required when particular action will occur.

  • Child User Invite Code Label

    Here you can change invite code popup label.

  • Child User Invite Code Empty Error

    Here you can change invite code textbox error message.

  • Child User Invite Code Used Error

    Here you can change invite code already used error message.

  • Child User Invite Email Available Error

    Here you can change invite code email availability error message.

  • Child User Invite Code Available Error

    Here you can change invite code availability error message.

  • Child User Invitation Success Message

    Here you can change child user successfully invite message.

  • Child User Delete Message

    Here you can change child user delete message.

  • Child User Resend Email Success Message

    Here you can change child user resend email successfully message.

  • Child User Refresh Invite Code Message

    Here you can change child user refresh invite code successfully message.

Email Template Tags For Group Membership

  • {ARM_GM_INVITE_COUPON_CODE}

    This email template tag is used to display invite code when parent user invite any child user.

  • {ARM_GM_INVITE_PAGE}

    This email template tag is used to display invite page link when parent user invite any child user.

  • {ARM_GM_PARENT_USER_USERNAME}

    This email template tag is used to display parent user username when child user receives notification of successfully signup as child user.

If any user signup as child user then user cannot purchase membership plans. And content permission will be inherited from their parent user.

Admininistrator can assign non group membership plan to member if group membership plan not available member account.

ARMember Group Membership does not possible with ARMember Multiple Membership.
