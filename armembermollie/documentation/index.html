This Add-on will add payment option for ARMember plan with one time payment & plan with subscription payment.

This Add-on will add support for the following Mollie payments methods to your ARMember Membership Payment Case:

  • Methods Supported with "One time Payment"

    • Credit Card
    • SOFORT banking
    • iDEAL
    • Bancontact
    • Banktransfer
    • SEPA Direct Debit
    • PayPal
    • Bitcoin
    • Belfius Direct Net
    • KBC/CBC Payment Button
    • paysafecard

  • Methods Supported with "Subscription Payment Case"

    • Credit Card
    • SOFORT banking
    • iDEAL
    • Bancontact
    • Belfius Direct Net
    • KBC/CBC Payment Button

This add-on will work with ARMember Version 1.6 or grater.

  • Easy Installation
  • Easy Setup of membership plan with mollie

  • setup membership plan for subscription payment with mollie

Installation & Activation

Steps For Installation And Activation

You can install plugin in following two ways. and ARMember - Mollie payment gateway plugin requires ARMember Plugin installed and active.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armembermollie.
  2. In your WordPress installation directory, upload armembermollie to wp-content/plugins/
  3. Activate the ARMember - Mollie payment gateway plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Mollie Setup Wizard

Steps to Setup Mollie For Your Website

Let's setup membership for your website :

  1. When plugin activated then mollie section will be display in ARMember -> General Settings -> Payment Geteways.

    Please see below screen.

  2. First of all need to activate the mollie payment gateway so you will be able to insert API Keys. For activation click on active button.

  3. If you do not have mollie account then create mollie account.

    For more information about create account and create web site on mollie account Click here.

  4. For API Key Login into your mollie account and click on setting page.

    If you do not have website profile then create a new web site profile.

    So in website profile you will see 'Live and Test API Keys' section.

  5. Click on 'Live and Test API Keys' section for API Keys.

    The section looks like below screen.

  6. Insert Live API key into Live API Key textbox and Test API Key into Sandbox API Key.

  7. Click on SAVE button.

  8. You need to set EUR currency in Default currency option on General Settings -> General page. Because mollie payment gateway support only euro currency.

  9. After setup mollie payment geteway you will be able to see mollie payment in your Plan + Signup Page in ARMember and set it for the plan.

    Please see below screen.

  10. Note : In Case of automatic subscription if payable amount will be 0 (zero), than user need to pay 0.10 EUR to start subscription.

    Because in mollie payment gateway, minimum transaction amount is 0.10 EUR is required to create subscription for customer.
