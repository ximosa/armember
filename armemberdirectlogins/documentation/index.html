Installation & Activation

Steps For Installation And Activation

You can install plugin in following two ways. and ARMember - Direct Login plugin requires ARMember Plugin installed and active.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armemberdirectlogins.
  2. In your WordPress installation directory, upload armemberdirectlogins to wp-content/plugins/
  3. Activate the ARMember - Direct Login plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Direct Login Setup Wizard

Steps to Setup Direct Login For Your Website

Let's setup ARMember - Direct Login for your website :

  1. When plugin activated then Direct Login menu will be display under ARMember

    Please see below screen.

  2. When you click on Direct Login then direct login users list will be display.

    Please see below screen.

    3. Actions

    For each row of grid there are some action options provided which are as following

    1. Copy link to clipboard

      Each direct login have unique link you can copy the link just one click on Copy link to clipboard Icon at right side.

    2. Modify Duration

      you can edit user expiration duration by just one click on Modify Duration Icon at right side.

      Please see below screen.

    3. Delete

      Each direct login can be deleted by just one click on DELETE Icon at right side.

    4. Active & Inactive

      You can ACTIVE or INACTIVE direct login using single switch.

  3. For Add new direct login click on Add Direct Login button.

    Please see below screen.

  4. For new user insert user email and select user role and select expiration hours or days.

  5. For exists user direct login please select user from user drop-down and select expiration hours or days.

  6. Then After click on save button. if you do not want to add user then click on cancel button.
