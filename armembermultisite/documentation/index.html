This add-on will allows to create, delete, activate/deactivate subsite to user on your ARMember membership setup via Member Network Site Addon.

This addon also work with multiple membership.

This add-on will work with ARMember Version 3.2.1 or grater.

Note: This addon only work if Enabled WordPress "Multisite Network" at your website.

ARMember - Member Network Site Addon Key Features:

  • Easy Installation
  • Easy Setup of subsite with a membership plan.
  • Easily manage subsite using Member Network Site addon.

Installation & Activation

Steps For Installation And Activation

You can install a plugin in the following two ways. and ARMember - Member Network Site plugin requires ARMember Plugin installed and active and also multisite enable.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armembermultisite.
  2. In your WordPress installation directory, upload armembermultisite to wp-content/plugins/
  3. Activate the ARMember - Member Network Site plugin through the Plugins menu from the WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Member Network Site Setup Wizard

Steps to Setup Member Network Site For Your Website

Let's setup ARMember - Member Network Site for your website:

  1. When you activate this plugin, you will be able to see a link "Manage Subsites" where you can view all the details of created subsites by members.

    Please take a look at below screen to get better idea.

  2. Admin can also activate/deactivate, view and delete subsite. please see below screen.

  3. You can configure a multisite option from the Manage Plans menu. please see below screen.
    Note: if your WordPress enable multisite then only you can see below option.

    If you want to allow users to create a subsite with a particular plan then enable Enable Multisite for Members option and enter the number of sites you want to allow. you can assign the role for subsite user by selecting Apply Role to Subsite Owner option.

Shortcodes for multisite addon:

  1. You can see Membership Shortcode block in Gutenberg editor. please see below screenshot.

  2. From the below screen you can add the shortcode for subsite creation form in your page.

    shortcode: [arm_site_creation form_id="101" arm_multisite_form_title="Add New Subsite" site_name="Site Name" site_title="Site title" enable_note_belt="true" site_limit_message="You have reached to limit of site(s) allowed to create." submit_button_text="Submit"]

    Parameter

    form_id: Form ID of form to apply form style to the subsite creation form.
    arm_multisite_form_title: Text that you want to display as title of the subsite creation form.
    site_name: Label for Site name input in form.
    site_title: Label for Site title input in form.
    enable_note_belt: Displays note as belt above the form title if it is true.
    site_limit_message: Message to be displayed while user reached at maximum limit to create a subsite.
    submit_button_text: Text to display on submit button in form.

  3. From the below screen you can add the shortcode for manage multisite in your page.

    Shortcode: [arm_manage_subsite form_title="Manage Subsites" records_per_page="10" no_record_message="There is no record of subsite" enable_note_belt="true" activate_button_label="Activate" deactivate_button_label="Deactivate" delete_button_label="Delete" membership_plan_label="Membership Plan" site_label="Site" status_label="Status" action_label="Action"]

    Parameter

    form_title: Text to display as title above the list of all subsites.
    records_per_page: Number of records to display per page.
    no_record_message: Message to be displayed when there is no records in the table.
    enable_note_belt: Displays note as belt above the title if it is true.
    site_limit_message: Message to be displayed while user reached at maximum limit to create a subsite.

In case of single membership

  1. After adding the multisite creation shortcode user can see multisite creation form at front-end side if user logged-in.

  2. After adding the multisite manage shortcode user can see multisite manage form at front-end side if user logged-in.

  3. If you try to create more subsite than allowed in the membership plan then it will give you an alert message. please see below screen.

  4. If you enable upgrade/downgrade option in plan setup and member is purchasing the plan which is under the condition of upgrade or downgrade then member's subsite will get deactivated.

  5. In case of canceling a plan or change the plan or remove the plan by admin, then member's subsite will get deactivated.

In case of multiple membership

  1. If you enable multiple memberships then you will get one another option in the site creation form. it indicates that under which plan you want to create a subsite.

  2. In case of multiple membership, If you activate any subsite then it will ask you to activate the subsite under a particular plan which is available to the user.

  3. If your subscription plan is suspended then your site will be deactivated automatically.

  4. If member renew plan then his subsite will not get affected, it will remain with the same status.

  5. If admin cancels the plan or change the plan or remove the plan then the related subsite of the plan will get deactivated.

  6. If the user has no plan then all subsite will be deactivated and will not even get activate option.
