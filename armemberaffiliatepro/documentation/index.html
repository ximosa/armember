If your system has activated "Affiliates Pro" plugin, then this add-on will allow you to integrate it with ARMember.

By using this combo of "ARMember + Affiliate Pro" user can get "commission" on sharing Referral of the plan - the plan for which 'Enable Affiliate Pro Referral' option is enabled.

In ARMember > Manage Plans > Add / Edit Membership plan you will get an option : "Enable Affiliate Pro Referral"

Important Note:This plugin will only work if ARMember plugin version is higher than 1.6 and AffiliatePro Version: 2.16.6 or higher.

This add-on will allow you to fully automates affiliate commissions for register users.

ARMember - Affiliate PRO Key Features:

  • Easy Installation
  • Easy Setup of membership plan with Affiliate PRO
  • Setup Referral Rate on Plan base, Flat rate, and Percentage
  • View Referrals and plan detail on referral page.

Installation & Activation

Steps For Installation And Activation

You can install plugin in following two ways. and ARMember - Affiliate PRO plugin requires ARMember Plugin installed and active.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armemberaffiliatepro.
  2. In your WordPress installation directory, upload armemberaffiliatepro to wp-content/plugins/
  3. Activate the ARMember - Affiliate PRO plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Affiliate PRO Setup Wizard

Steps to Setup Affiliate PRO For Your Website

Let's setup ARMember - Affiliate PRO for your website :

  1. When plugin activated then Affiliate PRO section will be display in ARMember -> Manage Plans -> Add / Edit Plan.

    Please see below screen.

  2. You will be able to enable and disable referrals plan wise.

  3. You can set referral rate in 'Fixed Rate' or 'Percentage' wise.

    Also you need to set this settings for each plan.

  4. You can see the new register user detail in Affiliates -> Referrals page

    Please see below screen.

  5. If you want to new user register as affiliate in Affiliates PRO. Then checked the 'Enable the user registration integration' checkbox.

    This option is available in Affiliates -> User Registration page.

    Please see below screen.
