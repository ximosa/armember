Installation & Activation

Steps For Installation And Activation

You can install plugin in following two ways. and ARMember - Square payment gateway plugin requires ARMember Plugin installed and active.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armembersquare.
  2. In your WordPress installation directory, upload armembersquare to wp-content/plugins/
  3. Activate the ARMember - Square payment gateway plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Square Setup Wizard

Steps to Setup Square For Your Website

Let's setup ARMember - Square payment gateway for your website :

  1. When plugin activated then square section will be display in ARMember -> General Settings -> Payment gateways.

    Please see below screen.

  2. First of all need to activate the Square payment gateway so you will be able to insert Application ID, and Access Token. For activation click on active button.

  3. If you do not have Square account then create it.

    For Live account click here.

  4. Where do I get the Application ID & Access Token?

    To obtain Application ID & Access Token, create new app in your account.

    Select your payment mode 'Sandbox' or 'Production'.

    After selecting payment mode you can see your Application ID & Access Token.

  5. To set up your callback URL(s) click on the OAuth tab in your account to be appear to the OAuth page. On the OAuth page you can see Redirect URL. Enter https://yoursite.com/?arm-listener=arm_square_api URL to same callback URL at ARMember -> General Settings -> Payment gateways -> Square -> callback URL.

    Please see below screen.

  6. After setup Square payment gateway you will be able to see Square payment in your Plan + Signup Page in ARMember and set it for the plan.

    Square Plan Setup

    In above screen display active payment gatewey including Square payment gateway.

  7. ARMember Square Addon supports payment with 'AUD', 'BRL', 'CAD', 'CZK', 'DKK', 'EUR', 'HKD', 'HUF', 'ILS', 'JPY', 'MYR', 'MXN', 'TWD', 'NZD', 'NOK', 'PHP', 'PLN', 'GBP', 'RUB', 'SGD', 'SEK', 'CHF', 'THB', 'USD', 'TRY', 'INR' Currency with Account.

Note: Square Payment Gateway doesn't support auto-debit Payment method.

Square payment gateway only works with secure sites (SSL).
