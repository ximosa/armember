Installation & Activation

Steps For Installation And Activation

You can install plugin in following two ways. and ARMember - Online Worldpay plugin requires ARMember Plugin installed and active.

New install via FTP

  1. Download the plugin and extract it. You will find a directory armember-worldpay.
  2. In your WordPress installation directory, upload armember-worldpay to wp-content/plugins/
  3. Activate the ARMember - Online Worldpay Addon plugin through the Plugins menu from WordPress admin panel.

New install via the WordPress Uploader

  1. Download the plugin.
  2. Log in to your WordPress admin panel
  3. Go to Plugins -> Add New -> Upload
  4. Click on "choose file" and browse the download package. When the plugin uploads successfully, click on Activate Plugin.

Online Worldpay Setup Wizard

Steps to Setup Online Worldpay For Your Website

Let's setup membership for your website :

  1. When plugin activated then Online Worldpay section will be display in ARMember -> General Settings -> Payment Geteways.

    Please see below screen.

  2. First of all need to activate the Online Worldpay payment gateway so you will be able to insert API Keys. For activation click on active button.

  3. If you do not have online worldpay account then online worldpay account.

  4. When you sign up in online worldpay then you will be able to get Service API key and Client key detail.

  5. After you Sign In account top right corner in display "ACTIVATE" Button. This button on clicking after your live mode activate.

  6. You can get api in Sign In online Worldpay account and go to settings menu page.

    Setting page in display API Keys menus. this menu on click then display API Keys.

    You can get Online Worldpay API Keys from here

  7. Insert Service API into Service key textbox and Client APi into Client key textbox in configration.

  8. If you want than you will be able to change below field label and description. If you will not set the label then default label will be display.

    • Credit Card
    • Expire Month
    • Expire Year
    • CVC

  9. if you enable "Display actual error returned from payment gateway" then it will display actual error from payment geteway while purchasing ARMember membership plan.

  10. Online Worldpay payment geteway supports GBP, EUR, USD, CAD, DKK, HKD, NOK, SEK and SGD currency. You can get all transaction in your native currency.

  11. Click on SAVE button.

  12. After setup online worldpay payment geteway you will be able to see online worldpay payment in your Plan + Signup Page in ARMember and set it for the plan.

    Please see below screen.

  13. To confirm transaction on Online Worldpay account, you can login with your Online Worldpay credential.
